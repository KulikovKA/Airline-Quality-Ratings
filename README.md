# Airline-Quality-Ratings

>Сравнительный анализ алгоритмов градиентного бустинга для решения задачи регрессии на данных авиаперевозок.

**Ключевые задачи:**
- Преобразование бинарной классификации в задачу регрессии (Feature Engineering).
- Сравнительный анализ производительности XGBoost, CatBoost и LightGBM.
- Автоматизация подбора гиперпараметров через TPE (Tree-structured Parzen Estimator).

## Методология

### 1. Подготовка данных
- **Целевая переменная:** Синтезирован непрерывный *Satisfaction Score (1.0–5.0)* на основе взвешенной суммы коррелирующих признаков ($r > 0.5$) с добавлением контролируемого шума.
- **Препроцессинг:** One-hot кодирование категориальных признаков; масштабирование (StandardScaler) применено только для линейных моделей.

### 2. Моделирование
Исследованные алгоритмы:
- **CatBoost Regressor** (Лучший результат)
- XGBoost Regressor
- LightGBM Regressor
- Random Forest Regressor
- Linear Regression (Baseline)

### 3. Оптимизация
Подбор гиперпараметров выполнен с помощью библиотеки `Hyperopt` (50 итераций на модель). Целевые метрики оптимизации: `MSE`, `MAE`.

### 4. Выводы
Приведана таблица по сравнению моделей по тестовым метрикам : MSE , MAE , R^2
